Step 1: Load the data file containing cell counts for treated and control cells.

 

(Click Load Example to the left to view a sample data file in the Data Tables tab.)

 

To calculate normalized growth rate inhibition (GR) values and corresponding metrics (GR50, GRmax, GEC50, GRinf, and Hill coefficient) based on cell counts measured in dose-response experiments using this online tool, users must provide a tab-separated (.tsv) data file in which each row represents a separate treatment condition and the columns specify the keys (variables) that define the treatment condition (e.g. cell line, drug or other perturbagen, perturbagen concentration, treatment time, replicate) and the measured cell counts (or surrogate such as CellTiter-Glo® or other readout). The experimentally measured cell counts that are required for GR metric calculation are as follows:


  • measured number of cells after perturbagen treatment (defined as ‘cell count’, x(c))
  • measured number of cells in control wells (e.g. untreated or DMSO-treated) on the same plate (defined as ‘control cell count’, xctrl)
  • measured number of cells from an untreated sample grown in parallel until the time of treatment (defined as ‘time 0 cell count’, x0)

 

The input .tsv file(s) must have the following column headers (first row of the file):

 

  • concentration = concentration of the perturbagen on which dose-response curves will be evaluated
  • cell_count = the measure of cell number (or a surrogate of cell number such as CellTiter-Glo® staining) after treatment
  • control cell count in a column labeled cell_count__ctrl  = the measure of cell number in control (e.g. untreated or DMSO-treated) wells from the same plate
  • time 0 cell counts in a column labeled cell_count__time0 = the measure of cell number in untreated wells grown in parallel until the time of treatment

 

All other columns will be treated as additional key variables (e.g. cell_line, drug, time, replicate).

 

Step 2: Select the grouping variables.

 

The key variables selected define on which variables the GR values will be averaged before fitting of the dose-response curves. By default, all variables are pre-selected, but, for example, replicates can be averaged by deselecting the proper column header. To deselect a variable, click on it in the grouping variables box and then press the "delete" or "backspace" key. You may repeat this with as many variables as you like.
